## P2 — 향후 확장

### 5. 지표 변동 알림
- 특정 지표가 임계값을 넘으면 이메일/푸시 알림
- 회원 알림 설정 UI 필요

### 6. 데이터 소스 확장
- OECD 수집기 (교육, 환경 지표)
- IMF 수집기 (국제수지, WEO)
- BEA 수집기 (미국 GDP 세부, 개인소득)
- 공공데이터포털 (한국 부동산, 교통 등)

### 7. 채팅 기능 고도화
- 3초 폴링 → WebSocket 전환 (실시간성 개선)
- 채팅 메시지 신고 (현재는 방 단위 신고만 가능)
- 채팅방 검색 기능

### 8. 관리자 기능 확장
- 사용자 강제 탈퇴 (현재는 목록만 조회)
- 신고 처리 내역 기록
- 수집기 실패율 모니터링 차트
